# Swahili DevHub

**Jukwaa la Waswahili Waanzilishi na Waendelezaji**

Swahili DevHub ni jamii ya mtandaoni inayolenga kuunganisha na kuwezesha Waswahili wanaopenda kujifunza na kuendeleza miradi ya programu. Tunatoa nafasi ya kujifunza, kushirikiana, na kukua pamoja kwa kutumia Kiswahili na Kiingereza kwa manufaa ya kila mshiriki.

## 1. Maelezo (Description)

Swahili DevHub ni jukwaa la jamii linalowezesha Waswahili waanfuseli na waendelezaji kushirikiana katika miradi ya programu, kushiriki rasilimali, na kudumisha mazingira rafiki ya kujifunza. Tunaendesha changamoto za coding, warsha, na hackathons kwa Kiswahili.

## 2. Vitu Vinavyohitajika (Prerequisites)

* Kadi ya malipo ya kimataifa (kwa wale wanaojiunga na GitHub ifuatapo)
* Akaunti ya GitHub
* Kupakua [Discord](https://discord.com)
* Ujuzi wa msingi wa Git na Markdown (kikao chetu cha mafunzo kinapatikana kwa [README ya GitHub](#))

## 3. Jinsi ya Kujiunga (Getting Started)

1. **Jiunge na Discord Server**:

   * Fungua: `https://discord.gg/swahilidevhub`
2. **Sanifu Wasifu Wako kwenye GitHub**:

   * Jisajili/jiengele GitHub: `https://github.com/join`
   * Jiunge na GitHub Organization: `https://github.com/SwahiliDevHub`
3. **Pata Rasilimali**:

   * Soma [Wiki ya Jamii](https://github.com/SwahiliDevHub/wiki)
   * Chukua changamoto za coding za wiki hii ([issues](https://github.com/SwahiliDevHub/community-challenges))

## 4. Matumizi (Usage)

* **Q\&A (Maswali & Majibu)**: Tumia channel `#somo` ndani ya Discord.
* **Onyesha Miradi**: Toa maoni na promozeni miradi yako kwenye `#projects`.
* **Warsha na Matukio**: Angalia `#events` kwa ratiba ya warsha zijazo.

## 5. Michango (Contributing)

Tuna-karibisha michango ya aina zote: code, makala (docs), tafsiri, na mentorship.

1. Clone repo:

   ```bash
   git clone https://github.com/SwahiliDevHub/README-template.git
   ```
2. Unda `branch` kichwa cha marekebisho:

   ```bash
   git checkout -b feature/jina-lako
   ```
3. Fanya mabadiliko, kisha tuma `pull request`.
4. Subiri uchunguzi (review); ukubaliwe, mtarajiwa ku-merge.

## 6. Kanuni za Tabia (Code of Conduct)

* Heshima na uvumilivu.
* Hakuna lugha ya chuki, ubaguzi, au udhalilishaji.
* Umuhimu wa ushirikiano na usaidizi.
